# Suggested Commands

- Install deps: `npm install`.
- Dev server: `npm run dev`.
- Production build/typecheck via Next: `npm run build`.
- Start production server after build: `npm run start`.
- Lint: `npm run lint`.
- Vercel build behavior: `npm run vercel-build`.
- Database push: `npm run db:push`; CI/forced push: `npm run db:push:ci`; GUI: `npm run db:studio`.
- Generate PWA/icons: `npm run generate-icons`.
- Windows shell note: use PowerShell syntax for local commands; quote paths with spaces; prefer tool `workdir` over `cd`.
- There is no `npm run test` script in current `package.json`; verify test command with user before claiming tests can run.
